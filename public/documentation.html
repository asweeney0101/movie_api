MyFlix API Documentation

/users (Post)

Allows a new user to regiser, automatically assigns them a unique ID.

/users (Get)

Returns a list of all users in the database.

/users/:Username (Get)

Returns the user that matches the requested username.

/movies (Get)

Returns a list of all movies in the database.

/movies/:title (Get)

Returns a JSON with information of the movie matching that specified title.

/movies/genres/:genreName (Get)

Returns a description of that genre.

/movies/directors/:directorName (Get)

Returns a description of the specified Director.

/users/:Username (Put)

Updates the selected user's information.

/users/:id/:movieTitle (Put)

Adds the movie typed at the end of the URL to the specified user's favorites list.

/users/:id/:movieTitle (Delete)

Removes the movie typed at the end of the URL from the specified user's favorites list.

/users/:id (Delete)

Allows the user to unregister from the system.
